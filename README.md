## Running Tests


### Git Commands

Clone Project Repo:

```bash
  git clone https://github.com/shantanu200/Feast-Factory.git
```

Move to Project Dir:

```
  cd feast-Factory
```

### FrontEnd Command

Move to client application

```
  cd client
```

Install Packages

```
   npm Install
```

Run application
```
    npm run dev / yarn run dev
```


### Backend Command

Move to client application

```
  cd server
```

Install Packages

```
   npm Install
```

Run application
```
    nodemon index.js
```


Then Check application url

```
  http://localhost:5173
```
